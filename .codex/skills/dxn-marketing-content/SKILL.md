# DXN Marketing Content

Create practical direct-selling content that can be used by a distributor or small team without creating compliance risk.

## Content Workflow

1. Identify the channel: website, WhatsApp, Facebook, TikTok/Reels script, brochure, training, event invite, or email.
2. Identify the audience: new customer, repeat customer, prospect, new member, active distributor, or team leader.
3. Define the single action the content should drive.
4. Use product/business facts only from official or user-provided sources.
5. Add a compliance pass for health, income, testimonial, and scarcity claims.

## Voice

- Be clear, warm, and concrete.
- Write for trust and repeat purchase, not pressure.
- Prefer simple product benefits and usage context over dramatic promises.
- For business-opportunity copy, emphasize support, learning, discipline, and effort.

## Reusable Content Types

- Product card: category, short description, usage context, price/contact action, disclaimer if needed.
- WhatsApp follow-up: greeting, context, one helpful detail, one question, low-pressure close.
- Distributor onboarding: first steps, training links, first customer follow-up, inventory/order guidance.
- Event invite: topic, date/time, audience fit, location/link, RSVP action.
- FAQ: direct question, short answer, source or caveat where applicable.

## Claims Guardrails

- Do not state or imply a product diagnoses, treats, cures, or prevents disease unless the user provides approved regulatory copy.
- Do not promise income, rank, bonuses, travel, or financial freedom as guaranteed outcomes.
- Avoid before/after claims unless the user provides compliant approved assets.
- Mark uncertain facts for verification instead of polishing them into confident copy.
